Pi Bubble single-folder runtime

How to run
1) Open terminal in this folder
2) Run: python main.py
   or on Windows: launch_bubble_app.bat

Notes
- Desktop webview mode needs nextjs-gui/out/index.html. Build it if missing.
- Load-cell module file hx711v0_5_1.py is not included in this repo; put it under library/hx711py if needed.
